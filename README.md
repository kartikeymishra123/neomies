# Neomies – Barber Booking App

🔗 Live Demo: https://trimtimesalon.netlify.app/

---

## 📌 Overview

TrimTime is a full-stack barber booking web application built using the **MERN stack**. It allows users to explore services, check real-time availability, and book appointments seamlessly with a smooth and modern UI experience.

---

## 🚀 Features

- 🧾 View all barber services with pricing  
- ⏰ Real-time slot availability  
- ❌ Prevent booking past dates & invalid time slots  
- 📅 Smart slot blocking based on service duration  
- 🧑 Simple booking flow (no authentication required)  
- 🎨 Premium responsive UI with modern styling  
- 🖼️ “Our Work” gallery section  

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- CSS (Custom Styling)
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Frontend: Netlify  
- Backend: Render
